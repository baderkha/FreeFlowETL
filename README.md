# FreeFlow ETL UI

Framework-first React + Material UI dashboard for a PipelineWise-style UI.

## Includes

- Dashboard page
- Connection creation/enrollment page
- Login/logout flow (UI placeholder)
- Settings page
- Source options: MySQL, PostgreSQL
- Target options: Snowflake, CSV, Amazon S3

## Run

```bash
npm install
npm run dev
```

## Notes

- This is UI scaffolding only.
- No backend integration or PipelineWise runtime logic is wired yet.
